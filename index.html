This site is a collection of some open source client-side bitcoin web apps I have built.

Feel free to fork any of these projects.

  • Orion.Ninja - Bitcoin transaction visualizer.
  • SegwitAddress.org - Segwit paper wallet app.
  • FastWallet - A serverless HD bitcoin web wallet I built for casual/throwaway use cases.
  • Explorer - Simple bitcoin block explorer.
  • BIP39 Generator - BIP39 mnemonic phrase generator that uses your mouse movements plus some CSPRNG for entropy.
  • MultiSig Paper Wallet - Multisig tool with Transaction Builder and Offline Key Signer.
  • Drag-Drop - Create private keys by drag and dropping a file or image.
  • Cold Signer - Tool to build and sign transactions offline.
  • Breev - A simple clone of the infamous bitcoin conversion site, preev.com
  • Buidljs - A playground for buidljs.
  • Mnemonic Recovery - A tool to brute force one missing word from a 12-word menmonic phrase.
